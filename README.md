# Countries App with Kotlin and GraphQL

### Disclaimer
This application is developed by forking the [GitHub repo](https://github.com/philipplackner/GraphQlCountriesApp) and following the [YouTube tutoria](https://www.youtube.com/watch?v=ME3LH2bib3g)l of Philipp Lackner.


### Installation 
1. Clone the GitHub repo: ```git clone git@github.com:jjose14-Jacob-Jose/GraphQlCountriesApp.git```
2. Open the project in Android Studio. Ensure you have GraphQL plugin installed. 

### Running the Project from scratch
1. In Android Studio Terminal, run the command ```./gradlew :app:downloadApolloSchema --endpoint='https://countries.trevorblades.com/graphql' --schema=app/src/main/graphql/com/plcoding/schema.graphqls``` (Terminal must be in the directory root).
